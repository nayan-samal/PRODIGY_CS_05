# Network Packet Analyzer - Task 05

This project implements a basic packet sniffer using Python and the Scapy library. It captures network packets and displays key information such as source IP, destination IP, and protocol.

## Features
- Captures live network packets
- Displays source and destination IP addresses
- Identifies protocol information
- Real-time packet monitoring
- Decode TCP/UDP layers
- Shows ports

## How It Works
The program uses the Scapy library to sniff network traffic. Each packet is analyzed and relevant details are printed to the console.
